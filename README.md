# ClinicPro — Clinic Management System

A clinic management web app built with **React + Firebase** for the **SMIT Ramadan Hackathon**.

## About

ClinicPro helps a clinic manage its daily operations through role-based dashboards. Each user logs in and gets access tailored to their role:

- **Admin** — manage doctors, patients, appointments, and staff
- **Doctor** — view assigned patients, appointments, and write prescriptions
- **Receptionist** — register patients and book appointments
- **Patient** — view own appointments, prescriptions, and medical records

Authentication is handled via **Firebase Auth** and data is stored in **Firestore**.

## Tech Stack

- React 19 + React Router 7
- Firebase Auth + Firestore
- Tailwind CSS 4
- Vite 8
- Deployed on Vercel

## Run Locally

```bash
npm install
npm run dev
```
